ABOUT
MobyLoader is a small Chrome extension that allows inline viewing of Mobypicture photos when a tweet is opened on Twitter.com.

INSTALLATION
Download the crx file from http://blaakmeer.com/mobyloader or package the files in the GitHub repository using this howto; http://code.google.com/chrome/extensions/packaging.html

USAGE
Click on a tweet that contains a moby.to link to open the tweet and the picture would load automatically.
